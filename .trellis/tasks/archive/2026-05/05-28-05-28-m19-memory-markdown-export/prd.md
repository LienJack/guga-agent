# M19 Memory Markdown Export

## Goal

Render active governed memory items into a human-readable Markdown projection.

## Requirements

- Group by scope and kind.
- Render active safe items only.
- Include confidence, importance, tags, and source refs.
- Bound item count and content length.
- Do not write files automatically.

## Out of Scope

- No parser/importer, automatic file mutation, retrieval changes, or prompt injection.
